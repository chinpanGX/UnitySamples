#### 簡単な2Dゲームを追加
__以下、ゲームの説明__
1. クリックした位置にボールが出現する
2. 2つのボールが衝突するとスコアが加算される
3. スコアの計算式 ```ボールが持つ値 + 加算補正値 * 2```
4. 3秒ごとに加算補正値は1ずつ増えていく
